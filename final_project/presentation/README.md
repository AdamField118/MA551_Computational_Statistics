# wpi-presentation-template

This is a minimal typst presentation template!

## What is in this repo?

- `presentation.typ` is your main entry point (your slides go here)
- `slides.typ` is a file for the individual style of each type of slide
- `theme.typ` is general style settings (colors, fonts)
- `bib.bib` is an example bibtex citation file
- `wpi.svg` and `background.png` are nice wpi images to make things look good
